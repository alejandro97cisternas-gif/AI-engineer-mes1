# 🤖 Asistente Gemini en Python

Este proyecto es un chat interactivo que utiliza la API de Google Gemini.

## 🚀 Características
* **Memoria de conversación**: La IA recuerda lo que dijiste antes.
* **Seguridad**: Uso de variables de entorno para la API Key.
* **Comentarios educativos**: Código totalmente documentado para estudio.

## 🛠️ Instalación
1. Clona el repositorio.
2. Crea un entorno virtual: `python -m venv env`.
3. Instala dependencias: `pip install -r requirements.txt`.
4. Crea tu archivo `.env` con tu clave de Google.
